Microsoft apagou seu enorme banco de dados faciais

06/06/2019 às 10:00h

Microsoft está apagando silenciosamente seu banco de dados MS Celeb, que contém mais de 10 milhões de imagens de cerca de 100.000 pessoas.

A Microsoft tem falado sobre o seu desejo de regular adequadamente a tecnologia de reconhecimento facial. O presidente da empresa, Brad Smith, apelou diretamente ao Congresso no ano passado para tomar medidas para administrar a tecnologia, que ele diz ter "amplas ramificações sociais e potencial para abusos". Tais são as preocupações da empresa que até bloqueou as vendas da tecnologia para as forças policiais da Califórnia. Agora, a Microsoft continua sua cruzada, apagando silenciosamente seu banco de dados MS Celeb, que contém mais de 10 milhões de imagens de cerca de 100.000 pessoas.

O banco de dados foi originalmente publicado em 2016, descrito pela Microsoft como o maior conjunto de dados de reconhecimento facial disponível publicamente no mundo, e usado para treinar sistemas de reconhecimento facial por empresas globais de tecnologia e pesquisadores militares. As pessoas cujas fotos aparecem no set não foram solicitadas por consentimento, mas como os indivíduos foram considerados celebridades (daí o nome do conjunto), as imagens foram retiradas da internet sob uma licença Creative Commons.

No entanto, o MS Celeb - que foi descoberto pelo pesquisador baseado em Berlim Adam Harvey - também continha imagens do que o FT.com chama de "indiscutivelmente particulares", como jornalistas de segurança e autores. Falando para o FT.com, Harvey - que dirige um projeto chamado Megapixels que revela detalhes sobre esses conjuntos de dados - também diz que mesmo que o MS Celeb tenha sido deletado, seu conteúdo ainda está sendo compartilhado pela web. "Você não pode fazer um conjunto de dados desaparecer. Depois de publicá-lo e as pessoas baixá-lo, ele existe em discos rígidos em todo o mundo", disse ele.

A exclusão ocorre pouco depois que o FT.com realizou uma investigação aprofundada sobre a tecnologia de reconhecimento facial e o papel da Microsoft nela. No entanto, a Microsoft explicou a exclusão do conjunto de dados para o FT.com como uma simples questão de protocolo. "O site foi destinado a fins acadêmicos", afirmou. "Foi administrado por um funcionário que não está mais com a Microsoft." Isso pode explicar por que a empresa não tem falado particularmente sobre o movimento - procedimento interno não pode realmente ser considerado o mesmo que um gesto ousado de boa vontade pública. No entanto, isso demonstra que a Microsoft está tão comprometida com a adesão legislativa quanto quer que todos sejam.
